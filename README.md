# Goxey 🏦

**Goxey** is a premium, Gen Z-focused financial extension for GXBank designed to make saving fun and spending intentional. By combining gamification, AI-driven insights, and social features, Goxey transforms the boring banking experience into an engaging journey.

---

## ✨ Key Features

1.  **🤖 AI Advice / Summary**
    Get personalized financial coaching with a twist. Our AI provides harsh yet humorous "Roast Mode" insights and summaries to keep your spending in check.

2.  **⚙️ Motor Friction**
    Stop impulse buys before they happen. Our variable friction checkout requires a 5-7 second hold for non-essential purchases, giving you time to think twice.

3.  **📊 Spending Visualization**
    Stop looking at boring numbers. Goxey translates your expenses into relatable real-world comparisons, like "This meal costs 100 packs of Maggi."

4.  **💰 Saving Automation**
    Make saving effortless with automated pockets. Set your goals and let Goxey handle the rest, ensuring you reach your milestones without the stress.

5.  **🎁 Gamification Blind Box Reward System**
    Turn financial discipline into a collection game. Earn rare GX Series characters and unbox "Hidden Edition" rewards as you hit your saving targets.

6.  **👥 Micro Social Circle & 3D Avatar**
    Join "Squad Mode" where your personalized 3D avatar reflects your financial health. Connect with friends and see who’s winning the saving game.

## 🛠️ Prerequisites

Before running the project, ensure you have the following installed:

*   **Flutter SDK**: [Install Flutter](https://docs.flutter.dev/get-started/install)
*   **Flutter & Dart Extensions**: Available in VS Code or Android Studio.
*   **Web Browser**: Google Chrome is recommended for web debugging.

---

## 🚀 Getting Started

Follow these steps to set up and run the app locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/TANJIANXIN13/Goxey.git
    ```

2.  **Pull the latest changes**
    ```bash
    git pull
    ```

3.  **Install dependencies**
    ```bash
    flutter pub get
    ```

4.  **Run the application**
    ```bash
    flutter run -d chrome
    ```
